Nested Classes

Table 1. Nested Classes
Type Description Example

Member Inner Class

  • defined at the member level

  • can be declared private, default, protected, public

  • can extend any class or implement interfaces

  • can be abstract or final

  • cannot declare static fields or methods

  • can access members of outer class, even static

 
public class Outer {
    protected class Inner {}
    public static void main(String[] args) {
      Outer outer = new Outer();
      Inner inner = outer.new Inner();
      // Ou
      Outer.Inner inner = new Outer().new Inner();
    }
}

→ Outer.class & Outer $Inner.class

Local Inner Class

  • defined within a method

  • no access specifier

  • cannot be declared static

  • cannot declare static or fields and methods

  • have access to all fields and methods of the enclosing class

  • access to local variables if only they are effectively final or final

Anonymous Inner Class

  • local inner without a name

  • are required to extend a class or implement an interface

 
public class AnonInner {
    abstract class SaleTodayOnly {
      abstract int dollarsOff();
    }
    public int admission(int basePrice) {
      SaleTodayOnly sale = new SaleTodayOnly() {
        int dollarsOff() { return 3;}
      };
      return basePrice - sale.dollarsOff();
    }
}

Static nested classes

  • defined at member level

  • can be instantiated without an object of the enclosing class

  • can be private

  • the enclosing class can refer to the fields and methods of the static nested class

public class Enclosing { static class Nested { private int price = 6; }

    public static void main(String[] args) {
      Nested nested = new Nested();
      System.out.println(nested.price);
    }
}
