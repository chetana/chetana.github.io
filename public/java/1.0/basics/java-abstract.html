Abstract class

  • Une classe est définie comme abstraite avec le mot clé abstract

  • Une classe abstraite ne peut pas être instanciée

    • Les classes abstraites sont utilisées lorsqu’une classe mère ne doit pas être instanciée

    • ne peut pas être final

  • Une classe abstraite n’est pas obligée de contenir de méthode abstraite

  • Si une classe contient une méthode abstraite, cette classe doit alors être déclarée abstraite

  • private or protected

  • Une classe abstraite A peut hériter d’une autre classe abstraite B. Dans ce cas, la première classe concrète héritant de A doit implémenter toutes les méthodes abstraites de A mais aussi celles de B

  • Si une classe abstraite intermédiaire fournit une implémentation d’une méthode abstraite d’une classe abstraite héritée, la classe concrète ne doit plus nécessairement refournir une implémentation de cette méthode

Abstract method

  • une méthode abstraite n’a pas de corps

  • implement an abstract method = sames rules as overriding

  • cannot be marked private of final
