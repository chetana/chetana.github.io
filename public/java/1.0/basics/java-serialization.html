Serialization

  • Permet de convertir un objet Java en mémoire en données stockées

  • Ecrire des objets ou des graphes d’objets directement vers des flux, que ce soit des fichiers ou en mémoire

Deserialization

Permet de convertir les données stockées en un objet Java

java.io.Serializable

  • une interface marker/tagging = ne possède aucune méthode

  • pour dire que les instances d’une classe peuvent aller dans des flux sérializés

  • la condition est que les classes de toutes les variables d’instance dans un objet sont sérialisables ⇒ NotSerializableException

  • transient : retirer la variable du processus de sérialisation

  • les membres statiques de la classe sont ignorés

  • SerialVersionUID (non requis) : permet de donner une version à notre classe et une exception sera levée si la version est mauvaise

https://www.youtube.com/watch?v=mCsdMGA7v9s&list=PLzzeuFUy_CngJCJfO9fVouqifqFAr-4Sf&index=33
