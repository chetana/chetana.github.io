Mots-clés en Java

Diamond operator <>

the types are being inferred from context with generic inference

abstract

cf Le modificateur abstract

assert

expression booléenne où l’on souhaite vérifier si quelque chose est vrai

break

exit the loop dependency on the condition that we are checking

class

declare a class

continue

the loop will bypass the part of code block that is below the continue keyword and continue with the next iteration

extends

permet de spécifier une superclasse éventuelle

final

  • rendre l’entité sur laquelle il s’applique non modifiable une fois qu’elle est déclarée

  • Une fois la variable déclarée final initialisée, il n’est plus possible de modifier sa valeur

  • Une méthode déclarée final ne peut pas être redéfinie dans une sous-classe

  • Une classe déclarée final ne peut pas être héritée

  • permet des optimisations par le compilateur

instanceof

  • a instanceof B : returns true if the reference to which a points is an instance of class B, a subclass of B or a class that implements B

  • For every x different than null, x instanceof Object returns true

  • compilation error if it knows that it is impossible

  • compilation checks only on a class, if interface it waits until runtime because a subclass could implement that interface without compiler knows it

native

  • indique que le code qui suit est écrit dans un autre langage que Java

  • limite la portabilité du code mais permet une vitesse d’exécution plus rapide

new

instancie un nouvel objet ou un tableau

null

une référence qui pointe vers un objet n’ayant aucune valeur

package

regroupe les classes Java selon ce qu’elles permettent de faire

static

  • variable : attribut commun à l’ensemble des instances d’une classe

  • méthode : appelée directement à partir du nom de la classe sans utiliser une instance

  • interface : //todo

super

used to access/call parent class members (variables and methods)

synchronized

assure l’accès à une méthode ou un bloc de code à un seul thread (les autres attendront)

this

  • indique l’objet courant

  • used to call the current class members (variables and methods)

throw

lance une exception

throws

utilisé pour la déclaration des exceptions

transient

retirer la variable du processus de sérialisation

volatile

  • permet de s’assurer que plusieurs threads concurrents puissent accéder à la dernière valeur de la variable

  • une variable volatile est accessible par plusieurs thread simultanément ! ⇒ ne pas confondre volatile et thread safe
