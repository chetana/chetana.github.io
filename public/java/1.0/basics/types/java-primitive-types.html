Complément à 2

Les int sont encodés sur 32 bits dans ce qu’on appelle le complément à 2. * 0b 0001 → 1 * 0b 0000 → 0 * pour -1, je dois prendre le complément de 1 (les 0 deviennent des 1 et les 1 des 0) et j’ajoute 1 ** 0b 1110 + 1 ⇒ 0b 1111 → -1

Nombres à virgule flottante = nombres réels

  • Les float sont encodés sur 32 bits et cet élément de 32 bits va être considéré comme un mot :

    • 23 bits : mantisse

    • 8 bits : exposant

    • 1 bit : signe de ce nombre flottant Virgule Flottante https://www.exploringbinary.com/why-0-point-1-does-not-exist-in-floating-point/ https://florian.cassayre.me/articles/float Convertir un nombre à virgule en binaire

  • utilisés lors de l’évaluation d’expressions nécessitant une précision fractionnaire

  • les calculs sont rapides mais non précis ! ⇒ dans le cas de calculs nécessitant une haute précision ou d’arrondi (transactions financières) utiliser BigDecimal !

  • float est une virgule flottante IEE 754 32 bits simple précision

  • double est une virgule flottante IEE 754 32 bits double précision

    • les nombres décimaux comme 0.1 n’existent pas en nombre à virgule flottante car la plupart de ces nombres ont une représentation binaire (base 2) infinie ⇒ un double n’aura jamais une représentation exacte de 0.1 !

Conversions - Cast

    • à condition qu’il y ait élargissement du domaine
