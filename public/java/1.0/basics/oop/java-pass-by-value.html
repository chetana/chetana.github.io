Passage par valeur

En Java, tout paramètre (que ce soit un type primitif ou un type objet) est passé par valeur !

Processus

  • Une copie des variables est faite et la fonction appelante manipule cette copie.

  • Si la copie est modifiée, la valeur originale est inchangée.

    • ex: assigner une nouvelle valeur primitive ou référence à un paramètre dans la méthode n’a pas d’impact sur l’appelant

// Types primitifs
public class PrimitiveByValueExample {

	public static void main(String... primitiveByValue) {
    	    int homerAge = 30;
    	    changeHomerAge(homerAge);
    	    System.out.println(homerAge); // 30
	}

	static void changeHomerAge(int homerAge) {
    	    homerAge = 35;
	}
}

// Assignation d'une nouvelle valeur référence
public static StringBuilder foo(StringBuilder a) {
    a = new StringBuilder("b");
    return a;
}

public static void main(String[] args) {
    StringBuilder s1 = new StringBuilder("s1");
    foo(s1);
    System.out.print(s1.equals("s1")); // true
    System.out.print(s1.equals("b")); // false
}

Même si la référence est passée par valeur, la copie pointe quand même vers un objet réel dans la mémoire Heap. * ex: si on modifie un des champs de l’objet ET que celui-ci est mutable, alors la valeur est également changée !

 
// pass a reference by value
public class ObjectReferenceExample {

	public static void main(String... doYourBest) {
    	    Simpson simpson = new Simpson();
    	    transformIntoHomer(simpson);
    	    System.out.println(simpson.name); // Homer
            // because the variable points to an object reference, the real object will also be changed !
	}

	static void transformIntoHomer(Simpson simpson) {
    	    simpson.name = "Homer";
	}

}

class Simpson {
	String name;
}

// pass an immutable object
public class StringValueChange {

	public static void main(String... doYourBest) {
    	    String name = "";
    	    changeToHomer(name);
    	    System.out.println(name); // "" because String is immutable : the fields in this class are final and can't be changed!
	}

	static void changeToHomer(String name) {
    	    name = "Homer";
	}
}

// pass a mutable object
 static class MutableObjectReference {
    public static void main(String... mutableObjectExample) {
      StringBuilder name = new StringBuilder("Homer ");
      addSureName(name);
      System.out.println(name); // Homer Simpson because StringBuilder is mutable
    }

    static void addSureName(StringBuilder name) {
      name.append("Simpson");
    }
  }
