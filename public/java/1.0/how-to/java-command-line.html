Lignes de commande basiques

Compilation d’un fichier

$ javac <className>.java
$ javac *.java

Exécution du programme

$ java className

Compilation avec les assertions (pouvant dégrader les performances de l’app)

$ java -ea Test

Récupérer la séquence hexadécimale du fichier.class

$ hexdump -C fichier.class
