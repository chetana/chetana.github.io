Moneta

JSR 354 : Java API for working with money and currency addresses the standardization of currencies and monetary amounts in Java Currency : Java 4 → ISO-4217 ⇒ lack a lot Goals : to provide an API for handling and calculating monetary amounts to define class representing currencies and monetary amounts as well as rounding to deal with current exchange rates to deal with formatting and parsing of currencies and monetary amounts

CurrencyUnit : models the properties of a Currency MonetaryAmount : concrete numeric representation of a monetary amount (always bound to a CurrencyUnit) Money.of() FastMoney.of() : alternative implementation that stores amounts in long fields (10-15 times faster than Money but limited by the size and precision of the long type) ATTENTION : FM and M are not the same instances !

<dependency> <groupId>org.javamoney</groupId> <artifactId>moneta</artifactId> <version></version> </dependency>
