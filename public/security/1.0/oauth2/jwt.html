JSON Web Token (JWT)

Json Web Token est un moyen d’encoder des données signées par un tiers.

Description

Il est composé de 3 parties :

  1. Header :

  2. Payload : expiration et rôle de l’utilisateur

  3. Signature : permet de valider le token

Format de transport de données signées

  • "Base64(Header)"."Base64(Payload)"."Base64(Signature)"

Toutes les informations mises dans un token JWT peuvent être potentiellement accessibles par tout le monde ⇒ ne jamais mettre de donnees confidentielles dans un JWT
